Importing a certificate from file into Mumble

  1. Select Configure -> Certificate Wizard
  2. ; 300px
  3. Select import certificate. Click Next
  4. Locate the certificate file. It should end in .p12 or .pfx depending on which browser exported the certificate.
  5. Enter the password for the certificate
  6. The certificate details should now be filled in. Make sure that it is the correct certificate. Click next.
  7. ; 300px
  8. Confirm that you want to change your certificate by pressing next.
  9. ; 300px
  10. Done! Press finish to close the guide.
