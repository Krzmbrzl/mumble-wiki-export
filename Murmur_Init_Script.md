Download the Script from
<https://raw.github.com/mumble-voip/mumble/master/scripts/murmur.init>

Rename it to /etc/init.d/murmur

Make the file executable :

`chmod +x /etc/init.d/murmur`

Update init levels

`update-rc.d murmur defaults 90`
