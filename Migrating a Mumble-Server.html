To migrate a Mumble-Server the following files are needed (on Debian in this example):

  • Database: /var/lib/mumble-server/mumble-server.sqlite
  • Configuration file: /etc/mumble-server.ini
  • Certificate related files: Check the following entries in your mumble-server.ini file and save those files:
    • sslca
    • sslcert
    • sslkey
