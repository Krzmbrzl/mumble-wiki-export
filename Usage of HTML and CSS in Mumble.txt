A subset of HTML and CSS can be used in the [[Mumble client]]; the full list of supported tags and properties can be found [http://qt-project.org/doc/qt-5/richtext-html-subset.html here].

Both HTML and CSS can be used in:
* The [[Murmur.ini#welcometext|welcome message]] of a server
* User comments
* Channel descriptions
* Text messages
