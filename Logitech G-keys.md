The G-keys available on many  [keyboards](https://en.wikipedia.org/wiki/List_of_Logitech_products#Gaming_Keyboards Logitech) can be used as keyboard shortcuts in Mumble 1.3.0 and later.

## First-time setup 
# Install the  [(LGS)](http://support.logitech.com/en_ca/software/gaming-software Logitech Gaming Software). If you already have it installed, make sure the version is newer than 8.55 -- If it is older, upgrade.
# Start LGS
# Start or restart mumble
